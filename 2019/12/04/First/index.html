##Vim使用技巧

#####技巧1 .命令

. 命令： 可以让我们重复上次的修改
x 命令： 删除光标下的字符
u 命令： 撤销上次的修改
dd命令： 删除一整行

#####技巧2 不要自我重复

>G 和<G 命令： 可以修改缩进
$命令： 移动光标到行尾
a命令： 在当前光标之后添加内容
A命令： 在当前行的结尾添加内容
C or c$命令： 表示修改当前行上光标后面的部分. 进入编辑状态

#####技巧3 以退为进

s命令： 删除光标下的字符，然后进入插入模式

#####技巧4 执行、重复、回退

f{char} or t{char}命令： 在行内查找下一处指定字符出现的位置，如果找到了就直接把光标移到那里
F{char} or F{char}命令： 在行内查找上一处指定字符出现的位置，如果找到了就直接把光标移到那里
；命令： 会重复查找上次f命令所查找的字符
,命令： 回退上次f命令查找

:s/target/replacement命令： 执行替换
：s/vivian/sky/ 替换当前行第一个vivian为 sky
：s/vivian/sky/g 替换当前行所有vivian为sky
n $s/vivian/sky/ 替换第 n 行开始到最后一行中每一行的第一个 vivian 为 sky
：n，$s/vivian/sky/g 替换第 n 行开始到最后一行中每一行所有 vivian 为 sky
n 为数字，若 n 为 .，表示从当前行开始到最后一行

#####技巧5 查找并手动替换

*命令： 查找光标所在的单词

cw命令： 会删除从光标位置到单词结尾的字符，并进入插入模式

#####技巧6 认识.范式

#####技巧9 构造可重复的修改

db命令： 删除从光标起始位置到单词开头的内容，但会留下最后一个字母
b命令： 把光标移到单词的开头
dw命令： 删掉整个单词
daw命令： delete a word
g~命令： 反转大小写
gu命令： 转换为小写
gU命令： 转换为大写
>命令： 增加缩进
<命令： 缩小缩进
=命令： 自动缩进

#####技巧13 在插入模式中可即时更正错误

Ctrl h命令： 删除前一个字符（同退格键）
Ctrl w命令： 删除前一个单词
Ctrl u命令： 删至行首

######技巧14返回普通模式

Esc 命令： 切换到普通模式
Ctrl [命令： 切换到普通模式
Ctrl o命令： 切换到插入-普通模式

zz 命令： 可以重绘屏幕
